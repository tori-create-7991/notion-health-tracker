# 体調管理アプリ - デザインドキュメント

## アプリ概要
Notionをデータベースとして使用し、月カレンダーで体調の推移を可視化するモバイルアプリ。

---

## 画面一覧

### 1. ホーム画面（カレンダー）
**目的**: 月単位で体調の推移を一目で確認できるメイン画面

**レイアウト**:
- ヘッダー: 月表示（例: 2025年12月）、前月/次月ナビゲーション、設定アイコン
- カレンダーグリッド: 7列×6行（日〜土）
- 各日セル: 日付番号、体調に応じた背景色

**コンテンツ**:
- 曜日ヘッダー（日月火水木金土）
- 日付セル（1-31日）
- 体調色による背景塗り分け
- 今日の日付ハイライト

### 2. 体調入力モーダル
**目的**: 選択した日の体調を記録・変更

**レイアウト**:
- モーダルオーバーレイ
- 選択日付表示
- 体調レベル選択（5段階）
- 保存/キャンセルボタン

**体調レベル**:
| レベル | ラベル | デフォルト色 |
|--------|--------|-------------|
| 5 | とても良い | #4ADE80 (緑) |
| 4 | 良い | #A3E635 (黄緑) |
| 3 | 普通 | #FACC15 (黄) |
| 2 | 悪い | #FB923C (オレンジ) |
| 1 | とても悪い | #F87171 (赤) |

### 3. 設定画面
**目的**: Notion連携設定とカラーカスタマイズ

**レイアウト**:
- ナビゲーションヘッダー（戻るボタン）
- Notion設定セクション
- カラー設定セクション

**コンテンツ**:
- Notion API Key入力フィールド
- Notion Database ID入力フィールド
- 接続テストボタン
- 各体調レベルのカラーピッカー

---

## ユーザーフロー

### 体調記録フロー
1. ホーム画面でカレンダーを表示
2. 記録したい日付をタップ
3. 体調入力モーダルが開く
4. 5段階から体調レベルを選択
5. 保存ボタンをタップ
6. Notionに保存 → カレンダーに色が反映

### 設定フロー
1. ホーム画面の設定アイコンをタップ
2. 設定画面に遷移
3. Notion API KeyとDatabase IDを入力
4. 接続テストで確認
5. 必要に応じてカラーをカスタマイズ
6. 戻るボタンでホームに戻る

## データ構造

### ローカルストレージ（AsyncStorage）
```typescript
interface Settings {
  notionApiKey: string;
  notionDatabaseId: string;
  conditionColors: {
    level1: string;
    level2: string;
    level3: string;
    level4: string;
    level5: string;
  };
}

interface HealthRecord {
  date: string; // YYYY-MM-DD
  condition: 1 | 2 | 3 | 4 | 5;
}
```

### Notionデータベース構造
| プロパティ | タイプ | 説明 |
|-----------|--------|------|
| Date | Date | 日付 |
| Condition | Number | 体調レベル（1-5） |

## インタラクション

### タップフィードバック
- 日付セル: opacity 0.7 + Haptics Light
- ボタン: scale 0.97 + Haptics Light
- 体調レベル選択: Haptics Medium

### アニメーション
- モーダル: フェードイン/アウト（200ms）
- 月切り替え: 横スライド（250ms）
